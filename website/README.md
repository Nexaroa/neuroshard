# NeuroShard Website

This is the official landing page for NeuroShard, built with React, Vite, Tailwind CSS, and Framer Motion.

## Getting Started

1.  **Install Dependencies**
    ```bash
    npm install
    ```

2.  **Run Development Server**
    ```bash
    npm run dev
    ```

3.  **Build for Production**
    ```bash
    npm run build
    ```

## Project Structure

*   `src/components`: Reusable UI components (Hero, Features, etc.)
*   `src/App.tsx`: Main landing page layout.
*   `tailwind.config.js`: Styling configuration.

## Deployment

The `dist` folder created by `npm run build` is ready to be deployed to any static host (Vercel, Netlify, AWS S3, etc.).
